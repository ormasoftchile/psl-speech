Solución psl-speech
-------------------
Implementa funciones de text-to-speech (TTS).

+ Dependencias
--------------
Interop.SpeechLib

+ Assemblies
------------
Pasamonte.Speech.Core
	- Define las interfaces para TTS
Pasamonte.Speech.SAPI
	- Implementa TTS utilizando Windows SAPI.

+ Referencia
------------
Al compilar, los assemblies son copiados en directorio assemblies general de Pasamonte.
